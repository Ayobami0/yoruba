# [Èdè Yorùbá](https://en.wikipedia.org/wiki/Yoruba_language)

An interpreted programming language.
Following Thorsten Ball's - Writing an interpreter in go

## Datatypes
- strings
- numbers
- boolean

```
jeki oruko je "ayobami"
# Translates to: let name = "ayobami"
```
